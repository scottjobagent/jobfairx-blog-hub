# JobFairX Blog Review Hub

**Open `index.html`** (or the GitHub Pages URL) — every blog draft lives there behind a toggle, with dev instructions in the first toggle.

Files: `index.html` = review hub · `*.md` = CMS-ready post source (publishing notes at bottom of each) · `jfx-components-snippets.html` = reusable coded components + CSS (add the CSS to the blog stylesheet once) · `chart-*.png` = chart images to upload with posts.

Standing rule: **no site screenshots in posts** — coded components only.

New posts are added to this same hub as new toggles, ~2 per week.
